About
Folder Watcher Windows Service continuously watches one or more folders for new files, file changes, modifications and deletions in real-time. When a change occurs, you can execute an application. It will run as a Windows service, so that you needn't login into windows. And it is a standalone application and need not .net or java JRE be installed.

Key Features

1. Watch folders for file changes, modifications, deletions and new files in real-time.
2. Watch local folders or network shares (including hidden shares).
3. Run as a Windows service, you needn't login to Windows.
4. Execute an application when an event occurs.
5. Include/exclude filtering on files being watched.
6. No need .net or java be installed.

System Requirements
1.You will need a Pentium 150 PC or higher, 16MB RAM or more. 
2. The operating system should be 32 bit or 64 bit Win95, Win98, Win2000, WinNT 4.0, WinME, WinXp, Win2003, Win Vista, Win 7, Win 8, Win 2008, Win 2012.
